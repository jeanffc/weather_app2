## Getting Started

### Prerequisites

Install Flutter (Windows, macOS, Linux):

- [Flutter Documentation](https://flutter.dev/docs/get-started/install)

### Running

Open some simulator:

- iOS

- Android

Install dependences:

```
flutter pub get
```

Run project:

```
flutter run
```
